```text
До сих пор для аннотации типов вы использовали простые типы данных, такие как:

    int
    float
    str
    bool
    другие стандартные типы Python'а

Но вы также можете использовать и более сложные типы. При этом у вас останутся те же возможности , что и до сих пор. Вот некоторые из них:

    UUID 
    datetime.datetime
    datetime.date
    datetime.time
    frozenset
    Decimal
    bytes

Стоит это иметь ввиду при разработке реальных приложений. Дополнительно можете прочитать про возможность применения Annotated из пакета typing (или из typing_extensions для Python ниже 3.9). 
```
